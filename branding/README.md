# Branding

Coloca aqui los recursos visuales del proyecto.

## Uso sugerido

* `logo/` para logos principales.
* `images/` para banners, fondos e imagenes auxiliares.
* `icons/` para iconos y variantes pequeñas.

## Recomendaciones

* Usar nombres descriptivos.
* Preferir PNG o SVG para logos.
* Mantener versiones livianas para la UI.
