# AI Context

Este proyecto utiliza documentación específica para proporcionar contexto, arquitectura y reglas de desarrollo a asistentes de IA.

Antes de realizar cualquier análisis, propuesta o modificación de código, leer los archivos ubicados en:

```text
ai/context/
ai/design/
ai/rules/
ai/skills/
```

## Orden de lectura recomendado

### 1. Contexto del proyecto

Leer:

```text
ai/context/project.md
ai/context/scope.md
```

Estos archivos describen:

* Objetivos del proyecto.
* Alcance funcional.
* Restricciones.
* Decisiones generales.

---

### 2. Arquitectura

Leer:

```text
ai/design/architecture.md
```

Define:

* Organización del sistema.
* Responsabilidades de cada módulo.
* Flujo general de funcionamiento.

---

### 3. Reglas

Leer:

```text
ai/rules/
```

Contiene:

* Reglas de comportamiento.
* Reglas de desarrollo.
* Criterios de calidad.
* Restricciones de implementación.

---

### 4. Skills

Leer:

```text
ai/skills/
```

Cada skill define el comportamiento esperado de un módulo o plataforma específica.

Ejemplos:

```text
youtube.md
```
